# @amlplugins/sendgrid-event-webhook

Native `@sendgrid/client` SDK re-exported under the `@amlplugins` namespace.

> SendGrid Event Webhook — deliverability, engagement, and bounce events POSTed to your endpoint.

## Install

```bash
npm install @amlplugins/sendgrid-event-webhook
```

## Use

```ts
import { /* named exports from @sendgrid/client */ } from "@amlplugins/sendgrid-event-webhook";
// or
import sdk from "@amlplugins/sendgrid-event-webhook";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
